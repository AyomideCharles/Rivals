# Rivals ![Flutter](https://img.shields.io/badge/Flutter-02569B?logo=flutter&logoColor=white) ![Dart](https://img.shields.io/badge/Dart-0175C2?logo=dart&logoColor=white) ![Platform](https://img.shields.io/badge/Platform-iOS%20%7C%20Android-lightgrey?logo=apple) ![Status](https://img.shields.io/badge/Status-Ongoing-pink)

Rivals is a football fan community app where supporters pick one club — locked forever — and join a global community of fans to post, banter, and connect across 6 major leagues.

# Tech Stack & tools

- Framework - Flutter (Dart)
- Auth - Firebase Authentication
- Database - Cloud Firestore
- Media - Cloudinary (images + video)
- State Management - Provider
- Video - video_player
- Fonts - Google Fonts (Space Grotesk + Archivo + JetBrains Mono)
- Icons - Iconsax
- UI - flutter_screenutil
- flutter_smart_dialog
